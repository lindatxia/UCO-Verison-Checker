How to Use

To get started with our application, have the link of the software agreement you are using at hand. Make sure to include the https://. First enter the name of the software (if you are entering for a specific agreement type, add the type into this name. For example, "asana-eula", and the link to the specific agreement you want to compare.


Next you will need to enter in the starting section and ending section of the document.


If you have exisiting software in the system that matches, the system will identify it for you and ask if you would like to compare the text.

If the software is a new software in the system, you are able to manually load in the old file that may exisit on your file system. Click Choose file to insert it in. Then click load selected file to load the file and then click Continue.
